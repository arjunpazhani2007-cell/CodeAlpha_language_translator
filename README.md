# CodeAlpha Language Translator

## Description
This is a simple Language Translation Tool built using Python, Streamlit, and Deep Translator. It allows users to translate text between multiple languages using Google Translator.

## Features
- Translate text between multiple languages
- Easy-to-use interface
- Fast translation
- Built with Streamlit

## Technologies Used
- Python
- Streamlit
- Deep Translator

## Run the Project
```bash
pip install -r requirements.txt
streamlit run app.py
```
